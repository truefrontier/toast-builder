# Toast Builder

## Installation

Install this package via Composer by added the package and the repository link:

```composer
"require": {
  "truefrontier/toast-builder": "dev-main",
}

"repositories": [
  {
    "type":"vcs",
    "url": "https://github.com/truefrontier/toast-builder"
  }
]
```

Then run:
```bash
composer update
```
